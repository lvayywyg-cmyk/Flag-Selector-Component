# 更新日志

本文件记录了 Flag Selector 组件的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.0.0] - 2024-02-02

### 新增
- ✨ 新增轻量级模式，支持按需加载完整国家列表
- ✨ 实现 SVG 国旗图片本地缓存（7天有效期）
- ✨ 添加 IP 检测本地缓存（24小时有效期）
- ✨ 后台 IP 验证机制，自动检测位置变化
- ✨ PC 端独立样式文件（flag-selector-pc.css）
- ✨ 智能国家排序，用户所在国家优先显示
- ✨ 完整的示例页面和交互演示
- ✨ 模块化导出支持（CommonJS 和全局变量）

### 改进
- ⚡ 优化初始加载性能，只加载基础国家数据
- ⚡ 减少内存占用，完整列表按需加载
- ⚡ 改进跨平台兼容性（SVG + Emoji 双重显示）
- 🎨 优化移动端和桌面端的响应式体验
- 📝 完善代码注释和文档

### 修复
- 🐛 修复 IP 检测失败时的默认国家显示
- 🐛 修复移动端弹窗在滚动时的定位问题
- 🐛 修复用户手动选择后仍进行 IP 检测的问题

### 文档
- 📚 添加完整的 README 文档
- 📚 添加详细的 API 文档
- 📚 添加快速入门指南
- 📚 添加贡献指南

### 变更
- 💥 **重大变更**：默认使用轻量级模式
- 💥 移除对外部依赖的依赖
- 💥 重构组件架构，分离 FlagDisplay 和 FlagSelector

## [1.0.0] - 2024-01-15

### 新增
- ✨ 基础国旗选择功能
- ✨ 250+ 国家和地区数据
- ✨ 搜索和过滤功能
- ✨ IP 地理位置检测
- ✨ 响应式设计（移动端/桌面端）
- ✨ 本地存储用户选择
- ✨ Emoji 国旗显示方案

### 改进
- 🎨 精美的 UI 设计
- ⚡ 高性能渲染
